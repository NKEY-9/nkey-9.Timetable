# 📚 대학교 시간표 웹페이지

## 실행 방법

**파일 실행:**
- `index.html` 파일을 웹 브라우저(Chrome, Edge, Firefox 등)로 열면 됩니다.
- 파일 경로: `시간표 앱/index.html`

**단계:**
1. 파일 탐색기에서 `index.html` 파일 우클릭
2. "열기" → 기본 웹 브라우저 선택
3. 또는 웹 브라우저를 열고 `Ctrl+O`로 파일 선택

---

## 기능 설명

### 1. 시간표 설정
- **1시간 간격**: 기본값 (9시~17시, 교시 단위)
- **30분 간격**: 세밀한 시간 구분 가능

### 2. 강의 추가
다음 정보를 입력하여 강의를 추가합니다:
- **강의명**: 과목 이름
- **강의실**: 수업 장소
- **교수**: 담당 교수
- **시작/종료 시간**: 수업 시간
- **전공과목**: 체크박스로 표시

### 3. 시간표 표시
- 월요일~금요일의 시간별 강의 표시
- 전공과목은 분홍색으로 강조 표시
- 시간 간격에 따라 자동으로 업데이트

### 4. 강의 목록
- 등록된 모든 강의를 카드 형태로 표시
- 각 강의 삭제 버튼 제공

## 수정 포인트 3가지

### 1️⃣ 수업 시간 범위 변경
**위치:** `script.js` 파일, 약 115번 줄 근처

```javascript
const startHour = 9 * 60;   // 9:00 (변경 가능)
const endHour = 17 * 60;    // 17:00 (변경 가능)
```

**예시:**
- 8:00 시작: `const startHour = 8 * 60;`
- 20:00 종료: `const endHour = 20 * 60;`

---

### 2️⃣ 색상 변경
**위치:** `style.css` 파일

**배경색 변경 (약 17번 줄):**
```css
body {
    background-color: #ffffff;  /* 흰색 */
}
```

**테두리색 변경 (약 27번 줄):**
```css
.container {
    border: 2px solid #000;  /* 검은색 */
}
```

**전공과목 강조색 (약 240번 줄):**
```css
.course-card.major {
    background-color: #fff5f5;   /* 배경색 */
    border-color: #d32f2f;       /* 테두리색 */
}
```

---

### 3️⃣ 강의 정보 필드 추가/삭제
**위치:** `index.html` 파일, 강의 추가 폼 섹션 (약 27~39번 줄)

**새 필드 추가 예시:**

```html
<!-- 강의 추가 폼에 추가 -->
<input type="number" id="credits" placeholder="학점" class="input-field">

<!-- script.js에서 addCourse() 함수 수정 -->
const credits = document.getElementById('credits').value.trim();
// ... 검증 추가 ...
const course = {
    // ... 기존 항목 ...
    credits: credits  // 새 항목 추가
};
```
