# 내 소개 카드 & 시간표 웹페이지

## 실행 방법

1. `index.html` 파일을 더블클릭하여 웹 브라우저에서 열면 됩니다.
2. 별도의 설치나 서버가 필요하지 않습니다.

## 주요 기능

### 1. 내 소개 카드
- 이름, 직업, 자기소개를 입력하면 실시간으로 카드 미리보기가 업데이트됩니다.

### 2. 시간표
- **1교시(9시)부터 8교시까지** 자동으로 생성됩니다.
- **시간 간격 설정**: 1시간 또는 30분 단위로 변경 가능합니다.
- 각 교시마다 다음 정보를 입력할 수 있습니다:
  - 강의명
  - 강의실
  - 교수
  - 전공 여부 (체크박스)
- 삭제 버튼으로 불필요한 강의를 제거할 수 있습니다.

## 수정 포인트 3가지

### 1. 내 소개 카드 기본값 변경
**파일**: `index.html`  
**위치**: 20~22번 줄  
예시에 사용된 "김철수", "학생" 등의 기본값을 자신의 정보로 변경하면 됩니다.

```html
<input type="text" id="userName" placeholder="이름을 입력하세요" value="김철수">
<input type="text" id="userJob" placeholder="직업을 입력하세요" value="학생">
```

### 2. 시간표 교시 개수 변경
**파일**: `script.js`  
**위치**: 71번 줄 (for 루프의 8)  
현재는 8교시까지 설정되어 있습니다. 이 숫자를 변경하면 교시 개수를 조절할 수 있습니다.

```javascript
for (let period = 1; period <= 8; period++) {
```

### 3. 기본 시작 시간 변경
**파일**: `script.js`  
**위치**: 73번 줄 (9 * 60)  
현재는 1교시가 9시(9 * 60분)로 설정되어 있습니다. 다른 시간부터 시작하려면 이 숫자를 변경하면 됩니다.
- 예: 8시부터 시작하려면 `8 * 60`으로 변경

```javascript
const startMinutes = 9 * 60 + (period - 1) * timeInterval;
```

## 주의사항

- 외부 라이브러리를 사용하지 않아 순수 HTML, CSS, JavaScript로만 작성되었습니다.
- 입력된 데이터는 새로고침하면 초기화됩니다. (로컬 저장 미지원)
- 최신 웹 브라우저(Chrome, Firefox, Safari, Edge 등)에서 정상 작동합니다.
